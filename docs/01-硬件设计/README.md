# 01-硬件设计

本分类按硬件子系统组织笔记。建议按顺序读：[设计维度与评价维度](00-设计维度与评价维度.md)（建立框架）→ [经典案例入门：LEAP Hand](<01-经典案例入门：LEAP Hand.md>)（用一个具体案例把框架讲透）→ [评价维度的量化指标与测试方法](02-评价维度的量化指标与测试方法.md)（把定性判断落到具体数字和测试方法）→ [技术路线地图：驱动、减速、传动](驱动-减速-传动/00-技术路线地图.md)（横向铺开五条技术路线的全景图谱）→ 具体子话题。

目前已有内容：

- **设计维度与评价维度**：整个"01-硬件设计"分类的框架页，讲清楚驱动介质/减速机制/传动路径等设计维度，和指端轻量化/力矩密度等评价维度之间的对应关系。
- **经典案例入门：LEAP Hand**：用 LEAP Hand 这一个主流案例，把八个设计维度和六个评价维度串联起来，并给出从这个基线过渡到其他方案的路径。
- **评价维度的量化指标与测试方法**：六个评价维度对应的可量化指标、测试方法、真实产品/论文数值，以及ISO 9283、YCB物体集这类可借用的正式/半正式标准。
- **驱动-减速-传动**：见 [技术路线地图：驱动、减速、传动](驱动-减速-传动/00-技术路线地图.md)——五条技术路线（关节内置驱动、腱绳驱动、连杆驱动、流体驱动、混合传动）的总览、对比表和决策逻辑，以及五篇深挖文档：路线1-关节内置驱动、路线2-腱绳驱动、路线3-连杆驱动、路线4-流体驱动、路线5-混合传动。

规划中、尚未开写的子话题（先占位，供未来扩展）：手指结构与自由度设计、传感器（触觉/力/位置）、材料与制造。
