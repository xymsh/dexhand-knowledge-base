# 方案家族：行星中等比 —— "够用党"基线

> 本文档从 [路线1：关节内置驱动](路线1-关节内置驱动.md) 拆分出来，是关节内置驱动路线里"减速比谱线"上中等比这一个点的独立深挖页。目前内容是从路线1原有段落迁移过来的，尚未按更细的议题清单展开，先占位。

## 一句话结论

行星减速器的减速比（约 200–350:1）是这个方案家族的关键数字：电机原生力矩只有约 0.02 N·m，经过约 300 倍放大后达到 1–4 N·m，满足手指关节所需；但代价是反向效率极低——这是这个方案家族无法做好接触力控制的根本原因。

## 代表产品：LEAP Hand

代表：**LEAP Hand**（CMU, RSS 2023），详见[经典案例入门：LEAP Hand](<../01-经典案例入门：LEAP Hand.md>)。这是本知识库选定的坐标系原点，六个评价维度都不极端，工程维护最突出。

两个字能解释它为什么成为主流：**够用**。对于绝大多数操作研究（RL 抓取、模仿学习、遥操作）来说，需要的不是精细力控，而是可靠的位置跟踪 + 可复现的硬件 + 快速上手。关节内置舵机把工程复杂度降到了极低——任何会写 Python 的研究者，拿到 LEAP Hand，两天内就能跑起来第一个实验。这种低门槛让它成为全球实验室共享算法基准的标准平台。

## 变体演化

- **LEAP Hand V2**：只改了驱动耦合这一个维度——从 16 个电机全驱动改成 8 个电机+差动关节刚度，往欠驱动方向挪了一格，换取更低成本更简单的控制，仍属于本方案家族。
- **LEAP Hand V2 Advanced**：跳出了这个方案家族——改成腱绳驱动+软硬混合外壳，相当于同时改了传动路径和结构材料两个维度，变成了[路线2：腱绳驱动](路线2-腱绳驱动.md)的新成员。
- 同家族友商：**Allegro Hand**（Wonik Robotics）——同样关节内置+全驱动，16 个 torque-controlled 关节，333Hz CAN 总线控制，是同一设计点的不同实现。
