## 测试方法与标准

- [ISO 9283 for Industrial Robots: What Accuracy Tests Actually Prove](https://www.brieflyindustrial.com/news/Industrial_Auto/Advanced_Collaborative_Robots/ISO_9283_for_Industrial_Robots_What_Accuracy_Tests_Actually_Prove.html) —— 工业机器人位姿重复精度/路径重复精度正式测试标准，可类比套用到灵巧手关节重复精度测试
- [The YCB Object and Model Set and Benchmarking Protocols](https://people.eecs.berkeley.edu/~pabbeel/papers/2015-RAM-YCB-object-set.pdf) —— 学界最常用的整手抓取/操作综合能力横向对比基准
- [A Review of Powered Backdrivability of Robot Actuators for Human-Robot Interaction](https://www.researchgate.net/publication/354232128_A_Review_of_Powered_Backdrivability_of_Robot_Actuators_for_Human-Robot_Interaction) —— 反向可驱动性测试方法综述
