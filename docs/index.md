# 灵巧手知识库地图

## 研究对象与目标

**研究对象**：高自由度仿人手（High-DOF Anthropomorphic Hand）——也就是我们平时说的"灵巧手"。相比工业场景常见的二指/三指夹爪，这类手通常有 4 指以上、10 个以上自由度，目标是像人手一样完成抓取、操作乃至精细操作（拧螺丝、翻书、系鞋带），是人形机器人、假肢、遥操作等领域的核心执行器之一。

**目标**：系统化梳理灵巧手从硬件设计到智能算法的完整知识体系，不只告诉你"有哪些方案"，更重要的是讲清楚"为什么是这样设计的"——每一个技术选择背后，都对应着成本、精度、力控、耐用性之间的具体权衡。读完这个知识库，希望你能做到两件事：

1. **能从零动手搭建**：借助 LEAP Hand、ORCA Hand 等开源方案，理解怎么用低成本零件搭出一台能用的灵巧手，并且清楚每个部件为什么这么选。
2. **能看懂商业方案**：面对 Tesla、Shadow、ORCA Hand、TESOLLO 等商业化产品，能看懂它们在成本、精度、力控、耐用性等维度上各自做了什么取舍，分辨优劣，而不是只会比参数表。

**这个知识库不是什么**：它不是某一款手的组装说明书，也不是某个厂商的产品测评。更准确地说，它是一张"设计空间地图"——把灵巧手拆解成驱动、减速、传动、感知、控制、算法等若干个独立维度，再看不同产品在这些维度上分别做了哪些选择，理解选择背后的物理约束和工程权衡。
