# 01-硬件设计

本分类按硬件子系统组织笔记。建议按顺序读：[设计维度与评价维度](00-设计维度与评价维度.md)（建立框架）→ [经典案例入门：LEAP Hand](<01-经典案例入门：LEAP Hand.md>)（用一个具体案例把框架讲透）→ [评价维度的量化指标与测试方法](02-评价维度的量化指标与测试方法.md)（把定性判断落到具体数字和测试方法）→ [经典方案家族与代表产品](03-经典方案家族与代表产品.md)（横向铺开八个方案家族的全景图谱）→ 具体子话题。

目前已有内容：

- **设计维度与评价维度**：整个"01-硬件设计"分类的框架页，讲清楚驱动介质/减速机制/传动路径等设计维度，和指端轻量化/力矩密度等评价维度之间的对应关系。
- **经典案例入门：LEAP Hand**：用 LEAP Hand 这一个主流案例，把八个设计维度和六个评价维度串联起来，并给出从这个基线过渡到其他方案的路径。
- **评价维度的量化指标与测试方法**：六个评价维度对应的可量化指标、测试方法、真实产品/论文数值，以及ISO 9283、YCB物体集这类可借用的正式/半正式标准。
- **经典方案家族与代表产品**：八个在设计维度空间里各自走到极端的方案家族（基线、谐波关节内置、准直驱QDD、腱绳远端、极端欠驱动、连杆欠驱动、气动软体、商用假肢），配真实产品和variant演化关系。
- **驱动-减速-传动**：见 [概念与方案地图：驱动、减速、传动](驱动-减速-传动/00-概念与方案地图.md)，以及方案 1：关节内置舵机、方案 2：腱绳远端电机、方案 3：准直驱 QDD、方案 4：气动驱动四篇深挖文档。

规划中、尚未开写的子话题（先占位，供未来扩展）：手指结构与自由度设计、传感器（触觉/力/位置）、材料与制造。
